== Description ==

Track Google Rankings v3.0 is a powerfull google position tracker plugin for your wordpress site.

= Features =

* Track unlimited number of keywords.
* Keyword discovery
* Export all data for analysis in an .xls file.
* Graphical interface for each keyword.

--- No special configurations are required.. simpley download and install ---

== Installation ==

Follow these steps to install the Google Keyword Tracker plugin:

1. Download track-google-rankings.zip to your pc.
2. Extract the archive and Upload the `track-google-rankings` folder to your `/wp-content/plugins/` directory.
3. Activate the Track Google Rankings plugin through the 'Plugins' menu in WordPress.
4. Visit -> Settings -> Track Google Rankings to configure the plugin.

The plugin will add two new menus to your WP Dashboard,

* *Tools -> Track Google Rankings* - view current rankings, add/remove keywords.
* *Settings -> Track Google Rankings* - modify plug-in settings
